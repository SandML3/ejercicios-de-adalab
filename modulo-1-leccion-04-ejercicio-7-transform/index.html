Teconologías web

El término tecnología web, hace referencia al conjunto de herramientas y técnicas empleadas en el proceso de comunicación entre diferentes tipos de dispositivos en la web, es decir, a través de internet. Dentro de la tencología web, podríamos direferenciar dos tipos principales; tecnologías frontend y tecnologías backend. Las teconogías frontend para el desarrollo web son aquellas que permiten la creación de interfaces de usuario y el establecimiento comunicaciones con el servidor. Entre ellas, podemos encontrar:

  1. HTML. El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos.
  2. CSS. CSS (siglas en inglés de Cascading Style Sheets), en español «Hojas de estilo en cascada», es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado.Es muy usado para establecer el diseño visual de los documentos web, e interfaces de usuario escritas en HTML.
  3. JavaScript. JavaScript es el único lenguaje de programación que funciona en los navegadores de forma nativa (lenguaje interpretado sin necesidad de compilación). Por tanto se utiliza como complemento de HTML y CSS para crear páginas webs y hacerlas interactivas.
